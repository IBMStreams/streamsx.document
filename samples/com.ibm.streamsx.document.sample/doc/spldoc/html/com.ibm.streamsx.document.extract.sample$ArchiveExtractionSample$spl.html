composite ArchiveExtractionSample

The sample demonstrates DocumentSource operator usage for extraction of embedded documents from rar archive.

The test archive file contains 4 files of different types (doc, pdf, xls, ppt): 

	helloWORD.doc      
	IBMInfoSphereStreams-QuickStartGuide.pdf 
	seat_plan.xls                           
	StreamsPresentation.ppt

DocumentSource operator returns four tuples (tuple per document). Each tuple includes both document content (text) as well as document metadata (file location, mimetype,etc).
